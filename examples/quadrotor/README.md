# Quadrotor Examples

Quadrotor trajectory tracking examples using MPPI variants.

- **`hover.py`** - Stabilize quadrotor at a fixed hover position
- **`circle.py`** - Track a circular trajectory at constant altitude
- **`custom_trajectory.py`** - Follow waypoint-based trajectories with cubic Hermite spline interpolation
- **`hover_comparison.py`** - Compare MPPI, SMPPI, and KMPPI on hover task
- **`figure8_comparison.py`** - Compare MPPI variants on aggressive figure-8 trajectory
- **`trajectories.py`** - Shared trajectory generation utilities (hover, circle, lemniscate, waypoints)
