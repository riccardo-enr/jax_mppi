# CUDA Examples

CUDA-accelerated MPPI examples and tests. Requires CUDA toolkit and GPU.

- **`pendulum_jit.py`** - JIT-compiled CUDA MPPI for pendulum control
- **`test_cuda_config.py`** - Test CUDA configuration bindings
- **`test_cuda_bindings.py`** - Test CUDA-to-Python bindings
- **`test_cuda_mppi.py`** - Test CUDA MPPI controller
